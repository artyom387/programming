Programming 3 final work

Grass - Կանաչը խոտն է, որը բազմանում է N ցիկլը մեկ, մահանում է երբ նրան ուտում են
GrassEater - Դեղինը խոտակերն է, որը բազմանում է N հատ խոտ ուտելուց հետո և մահանում է N հատ չուտելուց հետո, կամ երբ նրան ուտում են
Predator - Կարմիրը գիշատիչն է, որը բազմանում է N հատ խոտ կամ խոտակեր ուտելուց հետո և մահանում է N հատ չուտելուց հետո, կամ երբ նրան ուտում են
Lava - Նարնջագույնը լավան է, ուտում է իր կողքի 8 վանդակներում եղածներին և մահանում է նրա կենտրոնըիր կողքի լավաներին դաձնելով պատ
Hunter - Կապույտը որսորդն է, սպանում է գիշատիչին իրենից 1 կամ 2 վանդակ հեռավորության նաև կարող է մահանալ N հատ գիշատիչ չուտելուց հետո, կամ լավայից
